# Gemini

Gemini는 Google ecosystem과 강하게 연결된 multimodal assistant family입니다. Gemini Apps에서는 Deep Research, Canvas, Gems, Drive/Workspace 연동 같은 기능이 agent workflow로 이어집니다.

## 기능 축

| 기능 | Agent 관점 |
| --- | --- |
| Chat | 텍스트, 이미지, 파일 기반 질의응답 |
| Deep Research | 계획, 웹 조사, source synthesis |
| Canvas | 문서, 앱, slide, code를 작업면에서 작성/수정 |
| Gems | 특정 목적의 reusable assistant 설정 |
| Google Workspace integration | Drive, Docs, Gmail 등 개인/조직 context 활용 |
| Multimodal generation | image, video, audio 관련 작업면과 연결 |

## 잘 맞는 작업

- Google Drive/Workspace 안의 자료를 바탕으로 한 작업.
- 웹 기반 조사와 보고서 초안.
- multimodal 입력이 포함된 설명, 변환, 생성.
- 반복 목적별 assistant를 나누는 작업.

## 확인할 것

- Workspace 연결은 편리하지만 private data 경계가 중요합니다.
- Deep Research는 계획과 출처가 핵심이며, 최종 보고서는 사람이 다시 검토해야 합니다.
- media generation 기능은 사용량과 이용 가능 범위가 자주 변합니다.
